BioChallenges

About

BioChallenge is a continuous, time-based evaluation of prediction methods using public SPARQL endpoints. It allows to define challenges through providing a SPARQL andpoint and query which are expected to yield new associations over time.

We evaluate methods in different tasks such as protein function prediction, gene-disease association
